Registro de equipo

  1. Para Registrar un equipo Solo debe pulsar el boton del menu izquierdo NUEVO EQUIPO.

  2. Se desplegara el siguiente formulario que debe rellenar con valores válidos

    Lo primero que debe hacer es marcar el campo Nombre del equipo


    Luego la comunidad autonoma a la que pertenece

    Posteriormente el color de las prendas en Casa y de Visitante debe seleccionar alguno de los colores que estan agregados al sistema

    subir logo

    • Para subir el logo debe marcar el checkbox y se habilitara el boton subir logo, que le permitirá seleccionar el nombre del archivo con el cual desea registrar el logo

    • Por ultimo debe seleccionar la cantidad de jugadores y si posee o no patrocinador El equipo que esta registrando.
      Una vez finalizado solo debe presionar agregar, y si desea borrar todo el formulario solo con pulsar el boton borrar se eliminara todo el formulario
